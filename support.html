Frequently Asked Questions

How do I connect MIDI Expressionist to my Mac? +
You can connect via USB (IDAM) for zero-latency or Wi-Fi for wireless control. USB requires enabling your iOS device in Audio MIDI Setup. Wi-Fi requires both devices on the same network and creating a network session in MIDI Network Setup.

For detailed step-by-step instructions, check our Quick Setup Guide.
Which DAWs are supported? +
MIDI Expressionist works with any MIDI-compatible DAW including Logic Pro, Pro Tools, Cubase, Ableton Live, Studio One, Reaper, and more. It sends standard MIDI Control Change messages.
What iOS versions are supported? +
MIDI Expressionist requires iOS 12 or later and works on both iPhone and iPad with automatic orientation support.
How do I create custom controller layouts? +
Use the Edit Settings Mode to add, remove, and customize controllers. You can adjust sizes, positions, MIDI parameters, and save unlimited presets for different projects.
